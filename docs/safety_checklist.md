# 安全检查清单

## 上电前

- 电池组绝缘、极性、总压、单串电压一致性检查完成。
- 主电源串联合适保险或熔断保护。
- 急停开关可直接切断推进功率，且位置易触达。
- ESP32、传感器、ESC 信号地共地，电源路径清晰。
- 推进器周围无人员、绳索、工具和松动物件。

## 固件保护

- 上电默认未解锁。
- 解锁前 ESC 输出保持空闲脉宽。
- 油门变化有斜率限制。
- 遥控信号超时进入空闲油门。
- 电池低压进入限功率或空闲油门。
- 传感器异常进入保守状态。
- 语音控制不能解锁系统，只能在已解锁状态下输出受限低速目标。
- 语音控制必须在 ESP32 侧二次限幅，不能只依赖 Android App 限幅。
- 语音角度转向必须依赖 IMU 正常读取；IMU 异常、角度控制超时或蓝牙心跳丢失时必须取消并回空挡。
- 航向锁定必须依赖 IMU 正常读取；IMU 异常、蓝牙心跳丢失、锁定、急停或手动油门变化时必须取消。
- 航向锁定初始策略只允许低速限幅差速修正，并保留角度容差，避免 IMU 漂移导致持续抖动。
- `停止声控` 后 Android 端不得执行除 `开始声控` 之外的语音命令。
- 看门狗开启，重启后仍然保持未解锁。

## 岸上测试

- 先使用限流电源或串联保险测试。
- 单电机分别测试方向和油门范围。
- 双电机低油门联动测试。
- 记录电池线、ESC、接头、电机温度。

## 水下测试

- 静水低功率短时测试。
- 每次测试后检查进水、胶封、接头和温升。
- 逐步增加油门和持续时间。
- 保留测试日志，记录电压、电流、温度、故障和环境。
