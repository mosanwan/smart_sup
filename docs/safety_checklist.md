# 安全检查清单

## 上电前

- 电池组绝缘、极性、总压、单串电压一致性检查完成。
- 主电源串联合适保险或熔断保护。
- 急停开关可直接切断推进功率，且位置易触达。
- ESP32、传感器、ESC 信号地共地，电源路径清晰。
- 推进器周围无人员、绳索、工具和松动物件。

## 固件保护

- 上电默认未解锁。
- 解锁前 ESC 输出保持空闲脉宽。
- 油门变化有斜率限制。
- 遥控信号超时进入空闲油门。
- 电池低压进入限功率或空闲油门。
- 传感器异常进入保守状态。
- 语音控制不能解锁系统，只能在已解锁状态下输出受限低速目标。
- 语音控制必须在 ESP32 侧二次限幅，不能只依赖 Android App 限幅。
- 语音前进三档/四档只能作为固定档位目标，仍必须受声控功率限制和 ESP32 二次限幅约束；声控功率默认 `70%`，可配置到最高 `100%`，提高前必须按岸上限流和低功率水下测试逐级验证。
- Android TTS 播报开始到播报完成后 `1s` 内必须暂停 ASR 识别输入、丢弃麦克风采样并清空 VAD/ASR 缓冲，不得把 TTS 确认语音再次解析成控制命令。
- `快点` / `慢点` 只能按设置页微调步进调整当前目标，不能绕过手动解锁、声控功率限制或 ESP32 侧限幅。
- 语音角度转向必须依赖 IMU 磁力计正常读取；未锁航时 IMU/磁力计异常、角度控制超时或蓝牙心跳丢失必须取消并回空挡。
- 航向锁定中语音角度转向只能移动锁航目标，普通油门、档位和微调只能更新基础油门，不能取消航向锁定。
- 航向锁定必须依赖 IMU 磁力计正常读取；IMU/磁力计异常、蓝牙心跳丢失、锁定、急停、界面取消锁航或语音取消航向锁定时必须取消。
- 当前航向 `HDG` 不得再由陀螺仪 Z 轴无限积分生成；应使用磁力计 X/Y 直接计算，并做轻度环形低通滤波，避免静止时因零偏持续累加或因瞬时噪声导致锁航抖动。磁力计安装位置必须远离电机、电池大电流线和铁磁材料，并在低功率测试中验证干扰。
- 磁力计校准只能从 Android 设置页或 USB 串口维护命令进入；进入、保存或清除校准时 ESP32 必须立即锁定、左右油门回空挡并取消航向锁定。校准应在推进器安全、实际安装位置、低风险环境下进行。
- 航向锁定保留可配置角度容差、最大转向角度和空档最大反推，避免 IMU 漂移导致持续抖动或空档原地修正过猛；当前高响应测试版本允许最高 `70%` 差速修正，空档反推默认最多 `-70%`，手动 App 来源单侧输出可到 `100%`，只能按岸上限流、低功率、短时下水的顺序逐步验证。
- `停止声控` 后 Android 端不得执行除 `开始声控` 之外的语音命令。
- 看门狗开启，重启后仍然保持未解锁。

## 岸上测试

- 先使用限流电源或串联保险测试。
- 单电机分别测试方向和油门范围。
- 双电机低油门联动测试。
- 记录电池线、ESC、接头、电机温度。

## 水下测试

- 静水低功率短时测试。
- 每次测试后检查进水、胶封、接头和温升。
- 逐步增加油门和持续时间。
- 保留测试日志，记录电压、电流、温度、故障和环境。
